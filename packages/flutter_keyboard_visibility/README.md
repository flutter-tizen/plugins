# flutter_keyboard_visibility_tizen [![pub package](https://img.shields.io/pub/v/flutter_keyboard_visibility_tizen.svg)](https://pub.dev/packages/flutter_keyboard_visibility_tizen)

The Tizen implementation of [`flutter_keyboard_visibility`](https://pub.dev/packages/flutter_keyboard_visibility).

## Usage

This package is not an _endorsed_ implementation of `flutter_keyboard_visibility`. Therefore, you have to include `flutter_keyboard_visibility_tizen` alongside `flutter_keyboard_visibility` as dependencies in your `pubspec.yaml` file.


```yaml
dependencies:
  flutter_keyboard_visibility: ^6.0.0
  flutter_keyboard_visibility_tizen: ^0.1.0
```
